AI Checklists

Fast, practical AI checklists for safer adoption, vendor evaluation, resumes, and meetings. Built for busy operators.

Last updated: February 27, 2026

What this page is

AI Checklists are short, actionable lists you can run before or after using AI. Use them to evaluate vendors, polish resumes for ATS, review LLM safety, and run meeting-note workflows.

How to use this hub in 5 minutes

  1. Pick a checklist that matches your task (resume, vendor eval, safety, or meeting notes).
  2. Open the checklist page and work through the items.
  3. Use the linked playbooks and tools where they are referenced.

When to use a checklist

Use a checklist when you need a fast pass before or after an AI task. Before applying: run the Resume ATS Checklist. Before signing an AI vendor: run the Vendor Eval Scorecard. Before or after deploying an LLM: run the LLM Safety Review for prompt injection, data access, and hallucination. For meeting notes and follow-up: use the AI Meeting Notes Workflow. Each checklist is short; work through it once, then bookmark and reuse when the situation repeats. Fastest wins: resume before apply, vendor scorecard before sign, LLM safety before deploy. No login required. Each checklist links to related playbooks and tools. Use them as quick guardrails.
