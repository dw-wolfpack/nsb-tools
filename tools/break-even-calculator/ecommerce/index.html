Break-even Calculator for Ecommerce

Ecommerce has fixed costs (platform, marketing, salaries) and variable cost per unit (COGS, shipping, payment processing).

Last updated: February 27, 2025

What this tool does

The break-even calculator in Ecommerce mode shows how many units you need to sell to cover fixed costs. You enter fixed costs, variable cost per unit (COGS, shipping, fees), and price per unit. Use the link below to open the calculator. Educational only; not financial advice.

When to use it

  • Pricing products and checking if you can cover fixed costs
  • Comparing scenarios (e.g. different AOV or variable cost)

Inputs explained

  • Fixed costs per month – Rent, salaries, Shopify/subscription, ads.
  • Variable cost per unit – COGS, shipping, payment fees per order.
  • Price per unit – Average order value or per-SKU sell price.

How to use it

  1. Open the calculator (link below). Select Ecommerce mode.
  2. Enter fixed costs, variable cost per unit, price per unit. Click Calculate.

Common mistakes

  • Mixing monthly and annual numbers
  • Forgetting payment fees (2 to 3%) in variable cost

FAQ

What counts as variable for ecom?

COGS, fulfillment, shipping, payment processing per order.

Use AOV or per-SKU price?

AOV if you sell a mix. Per-SKU for single-product stores.

Is this financial advice?

No. Educational only.
