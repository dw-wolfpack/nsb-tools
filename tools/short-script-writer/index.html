Short-Form Script Writer

Generate a timed short-form script with beats + shot list + caption.

Last updated: February 27, 2025

What this tool does

Generates a 30-second short-form script with hook, problem, solution, CTA, shot list, and caption. Designed for Reels, TikToks, and Shorts. You enter a topic and get a full draft to edit and film.

When to use it

  • Planning a Reel, TikTok, or Short and need a full script fast
  • You have a topic but not a structure (hook, problem, solution, CTA)
  • Want a shot list to guide filming
  • Need a caption draft to post with the video

How to use it

  1. Enter your topic. Click Generate.
  2. Replace placeholders with your content. Use the shot list for filming.
  3. Copy the caption for your post. Use Regenerate for a different structure.

Common mistakes

  • Using a topic that is too broad so the script feels generic
  • Using the output verbatim without adding your own examples or voice
  • Ignoring the shot list and then missing key visuals
  • Skipping the hook; the first line should stop the scroll

Use it with AI

This tool is not AI-powered. You can use the results with an AI assistant in these ways:

  • Paste the script and ask for a shorter 15-second version or a longer 60-second version.
  • Ask an AI to suggest B-roll ideas for each shot in the list.
  • Combine with the AI Toolkit for content workflows.
